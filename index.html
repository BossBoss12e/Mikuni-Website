A simple, multipurpose Discord bot.

🤡 Utility

  • Run Run code and get results instantly! Credits: FalseDev.
  • calculate Calculate what you want!
  • Afk This will Set You A Global Afk.
  • Remind Me I will remind you whatever you tell me to.
  • Define heck the meaning of a word.
  • Weather Check the weather!.

👽 Moderation

  • Automod Configure automod for your server!
  • Antialts Configure Mikuni antialts system.
  • Ban This will ban the user from the server.
  • Unban This will unban the user from the server.
  • Warn This will warn the user.
  • lockdown Lock the whole server.
  • Delete Channel This will delete the text-channel mentioned.
  • Add Role This will add the role mentioned to the user.
  • Remove Role This will remove the role mentioned from the user.
